# NumTraceX

NumTraceX is a phone number intelligence and OSINT toolkit built in Python. It analyzes carrier, region, timezone, social presence, and spam reports.

## Features
- Parse phone numbers (carrier, country, timezone)
- Detect presence in social/spam DBs
- Password-protected access
- Modular design
